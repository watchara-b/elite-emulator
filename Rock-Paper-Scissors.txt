🛡️ 1. ประเภทเกราะ (Armor Types)เราจะแบ่งเกราะของยูนิตในเกมออกเป็น 4 ประเภทหลัก:เกราะเบา / ชีวภาพ (Light / Biological): ทหารราบ, สัตว์กลายพันธุ์, สายลับ (เคลื่อนที่ไว แต่ตัวบาง)เกราะกลาง / ยานยนต์คล่องตัว (Medium / Agile): รถหุ้มเกราะเบา, หุ่นยนต์ขนาดเล็ก, รถตุ๊กตุ๊ก (สมดุลระหว่างความเร็วและความอึด)เกราะหนัก / ป้อมปราการ (Heavy / Structure): รถถังขนาดใหญ่, หุ่นรบยักษ์, อาคารฐานทัพ (เชื่องช้า แต่มวลสารแน่นหนา)อากาศยาน (Airborne): โดรน, เรือบรรทุกเครื่องบินแปลงร่าง, ยูนิตบิน (ลอยตัวอยู่เหนือข้อจำกัดของพื้นดิน)
⚔️ 2. ประเภทอาวุธ (Damage Types)อาวุธแต่ละชนิดจะมี "ตัวคูณความเสียหาย" (Damage Multiplier) ที่ทำปฏิกิริยากับเกราะแต่ละประเภทต่างกัน:อาวุธสังหารบุคคล / พิษ (Anti-Infantry / Toxic): ปืนกล, เครื่องพ่นไฟ, สปอร์เห็ดพิษจุดเด่น: ยิงรัว ทำความเสียหายต่อเนื่อง (DoT) หรือเป็นวงกว้างชนะทาง: 🟢 เกราะเบา (ละลายทหารราบเป็นน้ำ)แพ้ทาง: 🔴 เกราะหนัก (ยิงรถถังแทบไม่เข้า)อาวุธระเบิด / กระแทก (Explosive / Blast): จรวด RPG, ปืนใหญ่, คลื่นน้ำจุดเด่น: พลังทำลายล้างเป็นกลุ่มก้อน (Splash Damage)ชนะทาง: 🟢 เกราะกลาง และ เกราะหนัก (แรงกระแทกทำลายเครื่องยนต์รบ)แพ้ทาง: 🔴 เกราะเบา (ทหารราบวิ่งหลบวิถีโค้งของจรวดได้ง่าย) / อากาศยานอาวุธเจาะเกราะ / พลังงาน (Armor-Piercing / Energy): เลเซอร์ความร้อนสูง, ดาบพลาสม่า, ปืนแม่เหล็กไฟฟ้าจุดเด่น: พลังโจมตีเป้าหมายเดี่ยวที่รุนแรงและแม่นยำสูงชนะทาง: 🟢 เกราะหนัก (เจาะทะลุเกราะหนาๆ ได้สบาย)แพ้ทาง: 🔴 เกราะเบา (Overkill เกินไป ยิงทหารราบทีละคนไม่ทัน)อาวุธต่อสู้อากาศยาน / EMP (Anti-Air / EMP): จรวดติดตามขีปนาวุธ, ปืนกลต่อสู้อากาศยานจุดเด่น: ล็อกเป้ายูนิตบนฟ้าโดยเฉพาะชนะทาง: 🟢 อากาศยาน (สอยร่วงอย่างรวดเร็ว)แพ้ทาง: 🔴 เกราะทุกประเภทบนพื้นดิน (ยูนิต AA มักจะโจมตีพื้นดินไม่ได้ หรือเบามาก)📊 ตารางสรุปการแพ้-ชนะทาง (Damage Multiplier Matrix)ประเภทอาวุธ (Damage) \ ประเภทเกราะ (เกราะเบา) มาหยุด Anubis Walker ทำให้หุ่นยักษ์ขยับไม่ได้ เปิดทางให้ตุ๊กตุ๊กยิงถล่มสถานการณ์จำลองที่ 2: ป้อมปราการปะทะป่าดิบชื้นสวิตเซอร์แลนด์ขับ Alps Fortress (เกราะหนัก / ปืนใหญ่ระเบิด) มาปิดล้อมบราซิลส่ง Jaguar Stalkers (เกราะเบา / อาวุธพิษชีวภาพ) เข้าไปลอบโจมตี -> สปอร์พิษยิงรถถังไม่เข้า (25%) แถมโดนปืนใหญ่ระเบิดใส่พื้นกระเด็นบราซิลเปลี่ยนแผน นำ Spore Launcher (ปืนใหญ่เชื้อรา -> จัดเป็นอาวุธคล้ายๆ ระเบิดชีวภาพผสมเจาะเกราะแบบ DoT) ระดมยิงข้ามกำแพงใส่เกราะหนัก ค่อยๆ กัดกร่อน Force Field ของสวิสจนพัง💡 กลไกเสริมเพื่อความลึกซึ้ง (Advanced Mechanic): Directional Armor (เกราะตามทิศทาง)เพื่อให้ผู้เล่นต้องใช้ทักษะการควบคุม (Micro) มากขึ้น คุณสามารถเพิ่มระบบ "เกราะหลังบาง" เข้าไปได้ครับ ยูนิตเกราะหนักอย่าง Alps Fortress หรือ Anubis Walker จะมีเกราะด้านหน้าหนามาก (รับดาเมจลดลง 30%) แต่ถ้าผู้เล่นฝ่ายไทยสามารถใช้ความเร็วของ Tuk-Tuk ขับอ้อมไปยิงด้านหลัง (Rear Armor) จะทำความเสียหายได้ 150% เสมอ ไม่ว่าอาวุธจะเป็นประเภทไหนก็ตาม!ระบบนี้จะทำให้รถตุ๊กตุ๊กของไทยที่ดูเหมือนจะสู้เลเซอร์ไม่ได้ มีโอกาสพลิกเกมได้ด้วยความเร็วและการยืนตำแหน่ง (Positioning) ครับ!
